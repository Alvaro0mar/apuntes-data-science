## Machine Learning

### Cambio de enfoque: Programación Tradicional vs Machine Learning

**Programación Tradicional**
Se tienen los datos de entrada y el programador crea el programa para producir la salida deseada.

```
Datos de entrada + Programa (creado por el programador) → Salida
```

**Machine Learning**
Se tienen los datos de entrada y salida, que son inputs para un algoritmo que nos creará el programa.

```
Datos de entrada + Datos de salida → Algoritmo → Programa (modelo)
```

> 💡 En otras palabras: en programación tradicional el humano escribe las reglas; en Machine Learning el algoritmo *aprende* las reglas a partir de los datos.
